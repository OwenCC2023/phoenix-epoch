Security System:

Main Goal: Measure quantitatively how much physical security the population enjoys for themselves and their property.

General Principles:
	- 0-100 scale. Base is 30.
	- Security is generated by buildings and effected multiplicatively by policies.
	- Magnitude of security generated by a given policy or building is governed by the government components and the national ideologies.
	- Like Stability, Security is generated and tracked at the province level.

System Effects:
	- Security creates a modifier between 0.5 and 1.5 that acts as a multiplier on Stability. Base Security should apply a 0.9 multiplier.
	- Security creates a modifier between 0.7 and 1.2 that acts as a multiplier on Literacy Bonus. Base Security should apply a 0.9 multiplier.
	- High Security modestly increases growth/turn (provincial).
