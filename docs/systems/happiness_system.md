### 9. Happiness system

**What:** `Province.local_happiness` — a 0–100 metric (base 50) computed fresh each turn as a static recompute (like Security, not incremental like Stability). Happiness measures how well the nation's active policies align with its ideology traits.

**Formula:**
```
raw_score     = Σ over active policies: HAPPINESS_POLICY_MATRIX[category][level][trait] × trait_weight
                  strong trait weight = 1.0,  weak trait weight = 0.5
trait_baseline = trait_effects["happiness_baseline_bonus"] + trait_effects["happiness_baseline_penalty"]
happiness     = clamp(BASE_HAPPINESS(50) + trait_baseline + raw_score × HAPPINESS_POLICY_SCALE(0.5), 0, 100)
```

**`HAPPINESS_POLICY_MATRIX`:** Generated by `tools/build_happiness_matrix.py` from `trait_policy_matrix.xlsx`. Scores run –3 (hard opposed) to +3 (core expression). `'D'` values in the xlsx (policy bans) are treated as 0 — bans are handled by `POLICY_BANS`. Only non-zero entries are stored. Regenerate after editing the xlsx:
```bash
cd backend
DJANGO_SETTINGS_MODULE=phoenix_epoch.settings.dev ./venv/Scripts/python.exe ../tools/build_happiness_matrix.py
```

**Downstream effects (all live):**
| Effect | At 0 | At 50 | At 100 |
|--------|-------|-------|--------|
| Worker Productivity | 0.8× | 1.0× | 1.2× |
| Growth multiplier | 0.9× | 1.0× | 1.1× |
| Stability recovery multiplier | 0.9× | 1.0× | 1.1× |
| Corruption resistance | stub | stub | stub |

**Worker Productivity** is applied in `building_simulation.py` as a per-province multiplicative factor on building output, stacked on top of the existing national `worker_productivity` flat bonus from modifiers:
```python
productivity_mult = (1.0 + worker_productivity) * province_happiness_mult.get(province_id, 1.0)
output = amount × effective_capacity × designation_mult × efficiency_mult × productivity_mult
```

**Migration destination weighting** (`economy/population.py`): Both starvation migration and economic migration weight destination provinces by `happiness_mult × security_mult` (where mult = `local_happiness / 50`, floored at 0.1). Unhappy or insecure provinces attract proportionally fewer migrants.

**National aggregate:** After all provinces are processed, `pool.happiness` is set to the mean of all province happiness values.

**Key constants** (`economy/happiness_constants.py`):
```
BASE_HAPPINESS = 50.0
HAPPINESS_POLICY_SCALE = 0.5
HAPPINESS_STRONG_WEIGHT = 1.0
HAPPINESS_WEAK_WEIGHT = 0.5
```
